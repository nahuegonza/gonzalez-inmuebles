# ADM-204 | Bandeja de solicitudes de "Vendé tu propiedad"

**Tipo:** Historia de Usuario
**Estado:** ❌ FALTA
**Descripción:** Como administrador, quiero recibir en una bandeja separada las solicitudes de propietarios que quieren vender, ya que requieren un proceso de tasación distinto al de una venta regular.

**Criterios de Aceptación:**
* **Escenario 1: Captura de datos de tasación** ❌
    * **Dado que** ingresa un nuevo lead desde el formulario de "Vendé tu propiedad".
    * **Entonces** el administrador debe visualizar en el detalle de este lead los datos específicos: Localidad/provincia, Tipo de propiedad y comentarios de tasación.
    * > **Falta:** El formulario de tasación (`tasacion.html`) simula el envío con un `setTimeout` de 1.8s y no persiste los datos en ningún lugar. No hay vista en el panel admin para ver estas solicitudes. El sidebar del admin tiene el link "Tasaciones" con badge "Nuevo" pero no lleva a ninguna vista implementada.

**Bugs / Pendientes:**
- ❌ No hay backend ni persistencia para capturar solicitudes de tasación.
- ❌ No hay vista separada en el admin para gestionar solicitudes de tasación.
- ❌ El badge "Nuevo" en el sidebar del admin es decorativo.
- ❌ El formulario de tasación es completamente decorativo.
