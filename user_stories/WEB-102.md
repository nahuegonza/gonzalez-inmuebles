# WEB-102 | Ordenamiento de resultados en el listado de propiedades

**Tipo:** Historia de Usuario
**Estado:** ✅ DONE
**Descripción:** Como usuario buscador, quiero poder ordenar los resultados del listado de propiedades para priorizar las opciones que se ajusten a mi presupuesto.

**Criterios de Aceptación:**
* **Escenario 1: Uso del selector de ordenamiento** ✅
    * **Dado que** el usuario se encuentra en una vista de listado de propiedades.
    * **Cuando** hace clic en el selector desplegable "Ordenar por".
    * **Entonces** se debe desplegar un menú con las opciones: "Menor precio" y "Mayor precio".
    * **Y cuando** el usuario selecciona una de las opciones, la grilla de resultados debe actualizarse dinámicamente reflejando el orden seleccionado.
    * > **Implementado:** El selector `#filterSort` en `properties.html` tiene las opciones "Menor precio" (`asc`) y "Mayor precio" (`desc`). La función `filterProperties()` en `main.js` aplica el sort al array y re-renderiza la grilla.
